javascript类型检测

发表于 2016-11-16 | 分类于 javascript |

js 作为一种动态语言，在运行的时候可以改变结构。在我们写函数的时候，往往需要对传入的数据进行一些类型检测，以确保程序运行到下面的安全性。接下来为大家介绍几种 js 中的类型检测的方法。

typeof

相信大家都很熟悉这个，它是一个操作符，就跟 ‘+’ , ‘-‘ 一样，所以使用的时候不需要添加括号。
typeof 返回一个字符串，指示未经计算的操作数的类型。

instanceof

和 typeof 一样，instanceof 也是一个操作符。
instanceof 用来测试一个对象在其原型链中是否存在一个构造函数的 prototype 属性。

instanceof 总结：

  • 不能判别原始类型
  • 可以判别内置对象类型
  • 可以判别自定义对象类型

Object.prototype.toString.call

这个方法使用了 Object 原型上面的 toString 方法，
该方法被调用的时候，会执行下面的操作步骤：

  1. 获取 this 对象的 [[Class]] 属性的值;
  2. 连接字符串 ‘[object’ + [[Class]] 属性的值 + ‘]’;
  3. 返回 连接后的字符串

[[Class]]是一个内部属性，原生对象和宿主对象(bom对象)都拥有该属性。

  • 可以识别原始类型以及内置对象类型
  • 不能识别自定义对象类型

constructor

constructor 是对象的一个属性，用于返回创建该对象的函数，也就是构造函数。
在 Javascript 中，每个具有原型的对象都会自动获得 constructor。
基本上所有的 Javascript 内置对象都具备 constructor 属性。
例如： Array, Boolean, Date, Function, Number, Object, String 等。
