发表于 2016-12-24 | 分类于 编辑器 |

Vim 编辑器

Vim 是一款 linux 自带的编辑器，相当于 Windows 上面的 记事本。 因为 linux 上面的操作一般都是在 终端(Terminal) 上，Vim 没有图形界面，所有的操作都是 用键盘来完成 。在安装 Git 的时候自带一个 Vim，下面简单介绍一下 vim 的特点和一些用法。

模式

Vim 编辑程序有三种操作模式，分别为 编辑模式， 插入模式 和 命令模式，当运行 Vim 的时候，首先进入编辑模式。

  • 编辑模式：跳转，移动光标，剪切复制粘帖，搜索，替换。
  • 插入模式： 插入字。
  • 命令模式： 打开，保存，重名名文件等。

命令

命令 效果
– 编辑模式
—- 跳转
k, j, h, l 移动光标 上，下，左，右
ctrl + f / b 文件中前移一页/后移一页
gg 将光标定位到文件第一行起始位置
G 将光标定位到文件最后一行起始位置
NG或Ngg 将光标定位到第 N 行的起始位置
——– 当前屏幕范围内移动（命令前可以加数字）
H 将光标移到屏幕上的起始行（或最上行）
M 将光标移到屏幕中间
L 将光标移到屏幕最后一行
——– 行内移动
w 右移光标到下一个字的开头
e 右移光标到一个字的末尾
b 左移光标到前一个字的开头
0 左移光标到本行的开始
$ 右移光标到本行的末尾
^ 移动光标到本行第一个非空字符串
—- 搜索匹配
/str1 正向搜索字符串 str1
n 继续搜索，找出 str1 字符串下次出现的位置
N 继续搜索，找出 str1 字符串上一次出现的位置
?str2 反向搜索字符串 str2
—- 替换和删除
rc 用 c 替换光标所指向的字符
nrc 用 c 替换光标所指向的前 n 个字符
x 删除(剪切)光标所指向的当前字符
nx 删除(剪切)光效所指向的前 n 个字符
dw 删除(剪切)光标右侧的字
ndw 删除(剪切)光标右侧的 n 个字
db 删除(剪切)光标左侧的字
ndb 删除(剪切)光标左侧的 n 个字
dd 删除(剪切)光标所在的行, 并且去除空隙
ndd 删除（剪切） n 行内容，并且去除空隙
——– 常用替换（会从编辑模式进入插入模式）
s 用输入的正文替换光标所指向的字符
S 删除(剪切)当前行
ns 用输入的正文替换光标右侧的 n 个字符
nS 删除(剪切)当前行在内的 n 行
cw 用输入的正文替换光标右侧的字
cW 用输入的正文替换光标到行尾的所有字符(同 c$ )
ncw 用输入的正文替换光标右侧的 n 个字
cb 用输入的正文替换光标左侧的字
ncb 用输入的正文替换光标左侧的 n 个字
c$ 用输入的正文替换光标到行尾的所有字符
c0 用输入的正文替换本行开始到光标的所有字符
—- 复制粘帖
p 将缓冲区的内容粘帖到光标的后面
P 将缓冲区的内容粘帖到光标的前面
—— 复制行
yy 复制当前行到缓冲区
nyy 复制 n 行到缓冲区
“+yy 复制 1 行到操作系统的粘贴板
“+nyy 复制 n 行到操作系统的粘贴板
—- 撤销和重复
u 撤销前一条命令的结果
. 重复最后一条修改正文的命令
– 插入模式
—- 进入插入模式
i 在光标左侧插入正文
a 在光标右侧插入正文
o 在光标所在行的下一行增添新行
O 在光标所在行的上一行增添新行
I 在光标所在行的开头插入
A 在光标所在行的末尾插入
—- 退出插入模式
ESC 或者 Ctrl+[ 退出插入模式进入编辑模式
– 命令模式
—- 打开、保存、推出
:e path_to_file/filename 已经启动的 Vim 中打开文件
:w 保存当前编辑的文件
:w file_temp 将当前文件另存为 file_temp
:q 在未作修改的情况下退出
:q! 放弃所有修改，退出编辑程序
:wq 保存并退出（先保存，再退出）
—- 行号与文件
:n 将光标移动到第 n 行
