Atom 隐藏了 .gitignore 忽略的文件

发表于 2016-12-21 | 分类于 编辑器 |

今天在学习 git 的时候发现，当文件夹内存在 .gitignore 文件的时候，里面相对应的文件在 Atom 左侧 tree-view 中不见了。有时候这不是我们想要的结果。
我们可以通过配置 config.cson 来控制这些文件的显示（/隐藏）。
ctrl + shift + p 打开命令面板，输入 app config 找到 Application: Open Your Config 来打开配置文件 config.cson。

tip：后来想想　.gitignore　里被忽略的文件一般都是一些依赖或者编辑器的配置啥的，一般那些配置都可以通过命令来修改，没必要去打开那些文件。
在 setting -> Package -> Core-packages -> tree-view 的设置界面也可以看到这个选项

最近在使用的时候发现 隐藏这些文件还是有用的。
在使用 ctrl + p 查找左侧打开的项目中的文件的时候。如果没有忽略 node_modules 这种大文件。查找的时候会很慢。
