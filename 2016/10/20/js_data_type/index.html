js数据类型

发表于 2016-10-20 | 分类于 javascript |

ES5中有6种数据类型，所有的值都这些类型之一。

  • Number
  • String
  • Boolean
  • Null
  • Undefined
  • Object

前面五种为简单数据类型（基本数据类型），后一种为复杂数据类型（本质上由一组无序的名值对构成）。
javascript 数据类型具有动态性。

typeof 操作符

因为 javascript 是松散类型的，所以需要一种手段来检测变量的数据类型——typeof 就是负责提供这方面信息的操作符。
对一个值使用 typeof 操作符可能返回的字符串：

  • ‘number’——数值
  • ‘string’——字符串
  • ‘bollean’——布尔值
  • ‘object’——对象或者null
  • ‘undefined’——未定义
  • ‘function’——函数

typeof 操作符的操作数可以是变量，也可以是数值字面量。 typeof 是一个操作符而不是一个函数，括号可用可不用。
typeof null 会返回object，因为 null 被认为是一个空的对象引用。

Number 类型

整数

八进制的字面量在严格模式(‘use strict’)下是无效的。
在进行算数计算时，所有八进制和十六进制表示的数值最终都将被转换成十进制数值。

浮点数值

浮点数值的最高精度是 17 位小数，在计算算数计算时精度远远不如整数。

数值范围

由于内存的限制，javascript能表示的数值有限。
最小值保存在 Number.MIN_VALUE 中，最大值保存在 Number.MAX_VALUE 中。
如果某次计算得到一个超过 Javascript 数值范围的值，这个数值将被自动转换成特殊的 Infinity 或者 -Infinity。
内置 isFinite() 函数。 当参数在范围之间时返回 true 。

NaN

NaN（Not a Number）是一个特殊的数值，用于表示一个本来要返回数值的操作数未返回数值的情况。（不抛出错误，不会停止代码执行）
特点：

  1. 任何涉及 NaN 的操作都会返回 NaN。
  2. NaN 与任何值都不相等。

内置 isNaN()函数。 接收一个参数，尝试将接收的值转换为数值。任何不能转换为数值的值会导致函数返回 true。

数值转换

有 3 个函数可以把非数值转换为数值： Number(), parseInt() 和 parseFloat();

Number()

Number() 函数作为包装函数，一般不建议直接使用，但是了解它的转换规则可以让我们更好的理解一些隐式转换的规则。
Number() 函数的转换规则：

  • Boolean，true 和 false 分别转换成 1 和 0。
  • 数值， 简单的传入与返回。
  • null， 返回 0。
  • undefined， 返回 NaN。
  • 字符串：
    • 只包含数字， 转换为十进制数值。
    • 只包含有效浮点格式， 转换为对应浮点数值。
    • 只包含有效十六进制格式， 转换为相同大小的十进制数值。
    • 空字符串， 转换为 0。
    • 包含以上格式之外的字符， 转换为 NaN。
  • 如果是对象， 先调用对象的 valueOf() 方法，依照上述方法，若结果是 NaN，则调用对象的 toString() 方法， 依照上述方法。

parseInt()

parseInt()函数在转换字符串时，

  • 忽略字符串前面的空格。
  • 如果第一个字符串不是数字字符或者负号或者是整数格式标识（’0x’表示十六进制）， 返回 NaN。
  • 解析第二个字符， 直到解析完所有后续字符或者遇到了一个非数字字符。

parseInt()支持 2 个参数，第二个参数表示转换时使用的基数（进制数）
建议任何情况下都明确指定基数。

parseFloat()

parseFloat()规则基本与 parseInt()相同， 解析到无效的浮点数数字字符为止。

String 类型

String 类型用于表示由零或多个字符组成的字符序列，即字符串。
字符串可以由双引号或者单引号表示。

字符字面量

String 数据类型包含一些特殊的字符字面量，也叫转移序列。

字面量 含义
\n 换行
\t 制表（tab）
\b 退格
\r 回车
\f 进纸
\ 斜杠
\’ 单引号
\” 双引号
\xnn 十六进制代码nn表示一个字符（256个）
\unnnn 十六进制代码nnnn表示一个Unicode字符

字符串的特点

js中的字符串是不可变的，一旦创建，它们的值就不能改变。
改变某个变量保存的字符串，要先销毁原来的字符串，然后再用一个新的字符串填充该变量。

转换为字符串

把一个值转换为一个字符串的两种方式

toString()

几乎每个值都有toString()方法，可以传递第二个参数来指定输出以任意有效进制格式表示的字符串值。
null 和 undefined 没有toString()

String()

String()可以将任何类型转换为字符串。
规则如下：

  • 如果有 toString()方法，调用该方法并返回相应的结果。
  • 如果是 null ，返回’null’。
  • 如果是 undefined, 返回 undefined。

Boolean 类型

Boolean 类型只有两个字面值： true 和 false 。区分大小写。

Boolean()

Boolean() 可以将一个值转换为对应的 Boolean 值。
转换规则如下：

数据类型 转换为 true 的值 转换为 false 的值
String 任何非空字符串 空字符串””
Number 任何非零数字 0 和 NaN
Object 任何对象 null
undefined / undefined

这些转换在理解控制语句的自动转换（隐式转换）非常重要。

因为if语句的判断里 期望 得到一个Boolen类型的数据，会对字符串 sMessage 自动执行 Boolean()，转换成了true。

Undefined 类型

Undefined 类型只有一个值， 为 undefined。
在使用 var 声明变量但未初始化时，这个变量的值就为 undefined。

Null 类型

Null 类型只有一个值， 这个值为 null。 null 值表示一个空对象指针。
null 常用来初始化一个用来保存对象的变量。

Object 类型

js 中的对象是一组数据和功能的集合。
创建 Object 类型的实例并为其添加属性和方法，就可以创造自定义对象。

Object 类型是所有它的实例的基础。 Object 类型所具有的任何属性和方法也同样存在于具体的对象中。
Object 的每个实例都具有的属性和方法：

  • constructor: 保存着用于创建当前对象的函数（构造函数）。
  • hasOwnProperty( propertyName ): 用于检测给定的属性在当前实例中（而不是实例的原型中）是否存在。
  • isPrototypeOf( object ): 用于检测传入的对象是否是传入对象的原型。
  • propertiIsEnumberable( propertyName ): 用于检测给定的属性能否使用 for-in 语句来枚举。
  • toLocaleString(): 返回对象的字符串表示，与执行环境的地区对应。
  • toString(): 返回对象的字符串表示。
  • valueOf(): 返回对象的字符串、数值或布尔值表示。

以上就是 ES5 中的数据类型啦。
